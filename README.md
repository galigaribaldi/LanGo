# LanGo
Repositorio para aprender a usar GO Golang, éste se encuentra dividido en 3 seccciones:  

* Básico:
    - Variables
    - Funciones
    - Ciclos
    - Condicionales
    - Estructuras
* Intermedio:
    - Paquetes
* Avanzado:
    - Hilos
    - Apuntadores
    - Conecciones a DB (Simples)
    - Creación de API's
    - ORM: conecciones con GORM (https://gorm.io/es_ES/docs/connecting_to_the_database.html)
